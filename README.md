# google-pubsub

## Setup

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Utilização

### API

```
python app.py
```

### Publisher

```
export GOOGLE_APPLICATION_CREDENTIALS=credentials.json
python publisher.py
```
